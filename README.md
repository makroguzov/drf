# Django REST framework

## Первая домашка

Для того что бы развернуть сервер, в корне проекта,
выполняем следующие команды:
```shell
./manage.py makemigrations   
./manage.py migrate 
./manage.py loaddata
./manage.py runserver
```

Перед этим все делом нужно понятное дло поставить все зависимости:
```shell
pipenv install
```
